# VeryBattery v1.0.0

VeryBattery는 장시간 전원 연결 환경을 위한 macOS 메뉴바 배터리 관리 앱입니다.

이번 릴리즈는 배포 가능한 첫 버전으로, 배터리 제어 CLI 번들링, privileged helper 지원, 실시간 배터리 모니터링, 한영 UI 다국어 지원이 포함되어 있습니다.

## 주요 변경사항

- `80% / 100%` 충전 한도 제어
- `80% -> 75%` 히스테리시스 기반 세일링 모드
- 강제 방전 모드
- 열 보호 모드
- 특정 앱 실행 시 자동 풀충전
- 외출 준비용 임시 풀충전 모드
- 배터리 잔량, 전원 상태, 온도 실시간 표시
- 앱 패키지 내부에 `battery` CLI 번들 포함
- privileged helper + XPC 기반 관리자 권한 제어 흐름 추가
- 영어/한국어 UI 다국어 지원

## 설치 방법

1. `VeryBattery.app` 다운로드
2. `/Applications`로 이동
3. 앱 실행
4. macOS가 실행을 막으면 `시스템 설정 > 개인정보 보호 및 보안`에서 허용
5. 처음 privileged 기능 실행 시 helper 승인 요청이 뜨면 승인

## 참고 사항

- 이 앱은 Mac App Store용이 아니라 일반 배포용입니다
- 처음 한 번은 관리자 권한 또는 helper 승인이 필요할 수 있습니다
- helper 승인 흐름은 macOS 버전에 따라 약간 다를 수 있습니다

## 설치 후 권장 확인 절차

- 메뉴바 팝업이 정상적으로 열리는지 확인
- 충전 한도를 `80%`로 변경
- `강제 방전 모드` 실행
- helper 승인 요청이 뜨면 승인 후 다시 시도
- 배터리 동작이 정상적으로 바뀌는지 확인

## 알려진 제약

- 실제 배터리 동작은 번들된 `battery` CLI와 대상 Mac 환경에 따라 달라질 수 있습니다
- 덮개를 닫은 clamshell 환경에서는 강제 방전 동작이 다를 수 있습니다
- 본격 배포 전에는 클린 Mac 테스트를 권장합니다

배포와 QA 절차는 `DEPLOYMENT.md`를 참고하세요.
